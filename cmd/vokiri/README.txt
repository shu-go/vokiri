# VOKIRI 

VOICEROID＋ 東北きりたん をコマンドラインから操作するソフトです。


# はじめに

付属の sample ファイルをダブルクリックしてみてください。
お手元の東北きりたんが、このソフトの紹介を始めます。

詳しい紹介は、このドキュメントや動画を参照してください。。


# 必要なもの

* VOICEROID＋ 東北きりたん
* コマンドラインへの少しの理解
* EXボイスなんて要らない、という強い心(VOKIRI では、EXボイスが使えません)


# とりあえず使ってみる

付属の launch をダブルクリックし、コマンドプロンプトを起動します。
(コマンドプロンプトが、VOKIRI のコマンドがあるディレクトリに移動した状態で起動します)

次のようにコマンドを入力します。

    > vokiri hello

※事前に VOICEROID が起動していない場合、自動的に起動してきます。 

VOICEROID が、コマンド vokiri に続けて入力した文字(上の例では hello)を読み上げます。


# 音声ファイルの保存

次の例のように、--record="〜" に保存したいファイル名(絶対パス)を指定します。

    > vokiri --record="%USERPROFILE%\Desktop\hello.wav" hello

読み上げる内容の前に --record を指定することに気を付けてください。
基本的に、読み上げる内容は、コマンドの最後に記述します。

また、指定できるパスには、絶対パスを指定する必要があります。
これは、VOICEROID が音声ファイルを保存するときには「VOKIRI のカレントディレクトリがどこか」を知らないためです。

## 無変更の音声ファイル保存のスキップ

VOICEROIDの設定で、音声ファイルの保存時に、テキストファイルも出力するようにしておきます。
(設定→音声出力設定→その他→テキストファイルを一緒に保存する)

加えて、--record="〜" を --record-once="〜" に変更します。

    > vokiri --record-once="%USERPROFILE%\Desktop\hello.wav" hello

これで、読み上げる内容が前回の保存時のものと同じ場合、VOICEROIDの操作を行わなくなります。


# 調音(フレーズ編集)

イントネーションの変更といった調音は、 ((テキスト｜調音内容)) の形式で、読み上げ内容の中に記載します。

    > vokiri "((テキスト | テ(pit:2)キス(pit:0.5)ト))"

コマンドプロンプトの半角記号の特別扱いを避けるため、
調音を使う場合には、読み上げる内容全体を半角の二重引用符で囲んでください。

事前にフレーズ辞書を使う設定にしておいてください。
(設定→日本語辞書設定→フレーズ辞書→フレーズ辞書を使用する)

上の例では、途中で pit つまり高さを変更しています。
変更は、 (pit:xx) を記述した後の音から有効になります。
コロン(:)の後に、VOICEROIDの画面にある調音の設定と同じ数値を指定します。

指定できるのは以下の4つです:

* 音量 : (vol:xx)
* 話速 : (spd:xx)
* 高さ : (pit:xx)
* 抑揚 : (emph:xx)

これ以外にも、VOICEROIDが認識できるフレーズ辞書用の表現が使えます。
一例をあげておきます。(ただしこれは間違っているかもしれません)

* ^ : 次の音のイントネーションを上げる
* ! : 次の音のイントネーションを下げる
* <R> : 疑問調にする


# 様々な利用手段

VOKIRI は、コマンドに文章を渡すことで、それを読む上げます。

そのほかにも、いくつか読み上げをさせる手段がありますので、紹介します。

## ファイルの読み上げ

    > vokiri < ファイル名

< の右側に、読み上げたいテキストファイルを記述します。

## ほかのコマンドの標準出力を受け取る

    > コマンド | vokiri
